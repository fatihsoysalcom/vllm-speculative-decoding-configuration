# vLLM Speculative Decoding Configuration

This example demonstrates how to configure vLLM to utilize speculative decoding. It shows how to initialize the vLLM `LLM` class by specifying a `draft_model` and a `speculate_length` parameter, which are crucial for enabling this inference acceleration technique. The example uses a small pre-trained model for quick demonstration.

## Language

`python`

## How to Run

1. Install vLLM: `pip install vllm`
2. Run the Python script: `python main.py`
